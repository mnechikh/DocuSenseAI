# **App Name**: DocuSense AI: A Multi-Tenant AI Knowledge Platform

## Core Features:

- Secure Multi-Tenant AI Knowledge Platform: Provides isolated environments for each small business, ensuring secure separation of authentication, relational database records, object storage, and vector indices. This foundational architecture enables businesses to confidently upload, process, and interact with their proprietary data.
- User & Role Management: Manages email/password authentication, user profiles (tenantId, userId, role), and provides tenant administrators with tools to invite, manage, and assign roles to users. Role-based permissions are enforced at the API level and designed for future extensibility.
- Robust Document Upload System: A user-friendly system supporting drag-and-drop uploads for PDF, DOCX, TXT, CSV, and XLSX files. Provides real-time status tracking including 'uploaded', 'processing', 'indexed', and 'failed', with additional details on failure reasons, processing timestamps, chunk count, and indexing status for comprehensive observability.
- Advanced Document Processing & Indexing: An asynchronous pipeline that extracts text from uploaded files, cleans and normalizes content, and applies configurable chunking (with overlap strategies). It generates embeddings for each chunk and stores them in a vector database alongside critical metadata for traceability: tenantId, documentId, filename, file type, upload timestamp, page/section, chunkId, and chunk position/order.
- Intelligent Chat Interface with Source-Grounded Answers Tool: A GPT-style chat interface enabling natural language queries against indexed tenant documents. It utilizes a sophisticated Retrieval-Augmented Generation (RAG) tool to retrieve top-K relevant chunks (always tenant-filtered), constructs prompts with system instructions and contextual information, and generates responses. The system is engineered to provide answers strictly grounded in the indexed tenant data, explicitly state if information is not found ('I don't have enough information'), and prevent hallucinations. Responses include precise citations to source documents (document name, page/section), with clickable links to a basic document section preview.
- Comprehensive Admin & Management Panel: A dedicated interface for tenant administrators, providing capabilities to upload, view status, and delete documents, manage users within their tenant, and access basic system settings. It centralizes control over tenant-specific data and user access.
- Granular Role-Based Access Control (RBAC): Enforces strict access policies at the API layer: 'Admin' users have full access (upload, delete documents, manage users, view all documents, chat), while 'User' roles are limited to chat interaction with allowed tenant documents and viewing citations. This ensures secure and controlled access to data and functionalities, ready for future expansion with custom roles.
- Persistent Chat History & Session Management: Enables users to save and access previous chat conversations, with message persistence and tenant-scoped history. Users can rename and delete chat sessions, ensuring a continuous and organized interaction experience.
- Robust Security, Data Privacy & Audit Logging: Implements secure file handling, role-based authorization on every API endpoint, and comprehensive audit logging for all critical admin actions and data access events. This ensures the highest standards of data privacy and security across the multi-tenant architecture.

## Style Guidelines:

- Primary color: A sophisticated, deep ocean blue (#2C4F7D) reflecting trust and depth of knowledge, appropriate for a business intelligence application. HSL(210, 50%, 35%).
- Background color: A clean, desaturated light blue (#ECF2F6) provides a serene canvas for content, maintaining readability and a professional appearance. HSL(210, 15%, 95%).
- Accent color: A vibrant teal (#4CCCD0) chosen from an analogous hue on the color wheel to inject energy and highlight interactive elements. HSL(180, 70%, 55%).
- Body and headline font: 'Inter' (sans-serif) for its modern, clean, and objective aesthetic, ensuring optimal readability and a polished, professional feel throughout the application.
- Utilize a consistent set of minimal and clear line icons that provide intuitive visual cues for actions and navigation, aligning with a 'simple interface' objective.
- A responsive, content-first layout featuring a consistent left-hand navigation, a central workspace for chat interactions or document viewing, and an optional right-side panel for source citations or document context. Admin-specific pages will be clearly separated from the main user workspace to maintain focus and clarity.
- Implement subtle, purposeful, and restrained animations for transitions, loading states, and status updates, providing immediate visual feedback that reinforces reliability without being distracting or flashy.
